Overview

The Objective

This project builds an intelligent customer support chat agent that handles user queries conversationally, detects when a user wants a callback, collects their name and phone number through natural conversation, validates the data, and automatically triggers an outbound AI voice call via VAPI — moving the user from chat to voice with zero human involvement.

The Challenge

Bridging Chat and Voice Automatically

The critical challenge was making the transition from chat to voice feel seamless and reliable. The AI needed to detect callback intent from natural language, guide the conversation to collect only the missing fields without being repetitive, validate phone format correctly, and only trigger the VAPI call once all required data was confirmed — all while maintaining a warm and professional support persona throughout.

The Solution

Technical Strategy

The n8n Chat Trigger receives incoming messages. A LangChain AI Agent powered by OpenAI classifies intent as either 'general' or 'call_request' and follows a guided data collection flow, asking only for missing fields. The agent produces dual output — a user-facing reply and a backend JSON block containing the captured fields and a readyForCall boolean. A Parser Code Node separates the two, a Validation Node checks completeness, and when all fields are confirmed, an HTTP Request Node fires the VAPI outbound call API with the assistantId, phoneNumberId, and captured phone number. Conversation memory is maintained throughout the entire session.

Key Results

Impact

  • Chat-to-voice automation with zero human escalation
  • Structured JSON intent extraction from natural conversation
  • Phone number validation before call trigger
  • Full conversation memory maintained across session
