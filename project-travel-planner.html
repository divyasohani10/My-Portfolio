Travel Planner
AI

n8n OpenAI Telegram Itinerary Generation
Overview

The Objective

This project automates personalized travel planning through a Telegram bot. A user simply sends a message like 'Plan a 5-day Paris trip' and receives a complete, structured itinerary covering daily activities, hotel recommendations, and budget estimates — generated by GPT-4o and delivered instantly back to Telegram. No forms, no travel agents, no manual research.

The Challenge

Generating Useful, Structured Itineraries

Generic travel suggestions are not useful. The challenge was prompt engineering GPT-4o to produce genuinely organized, day-by-day itineraries that include specific attractions, realistic timing, accommodation tiers, and budget breakdowns — all tailored to the user's destination, duration, and preferences extracted directly from their natural language message.

The Solution

Technical Strategy

A Telegram bot serves as the user interface, capturing natural language travel requests. n8n receives the message via Telegram trigger and passes the full request to GPT-4o with a structured system prompt designed to extract destination, duration, and travel style, then output a clean day-by-day itinerary. The response is formatted in readable Telegram markdown and sent back to the user within seconds. The system handles vague inputs gracefully and can generate plans for any destination worldwide.

Key Results

Impact

  • Full 5-day itinerary generated and delivered in under 10 seconds
  • Natural language input — no forms or structured data required
  • Handles any destination worldwide
  • Hotel, activity, and budget recommendations in one response
