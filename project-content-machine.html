Overview

The Objective

This project automates the entire LinkedIn content pipeline from idea to published post. The system generates fresh post ideas daily, avoids repeating past content, writes full post copy using GPT-4o, generates matching visuals using OpenAI image API, stores everything in Google Sheets, and auto-publishes to LinkedIn twice a day — completely hands-free.

The Challenge

Avoiding Repetition at Scale

The biggest challenge was building a system that stays creative over time without repeating itself. As the idea bank grows, the AI needs to actively compare against all previous content and generate genuinely new angles — while maintaining a consistent brand voice and visual style across every post.

The Solution

Technical Strategy

n8n orchestrates a two-schedule pipeline. At 5AM, the content generation workflow fetches past ideas from Google Sheets, passes them to an AI agent as context, generates a new idea with title, copy, and image prompt, generates the image via OpenAI, stores it in Google Drive, and saves everything to Sheets with status 'review'. At 4PM, a separate posting workflow fetches ready posts, uploads the image to LinkedIn, publishes via LinkedIn API, and updates the status to 'posted'.

Key Results

Impact

  • Fully automated daily LinkedIn presence
  • Zero manual content creation effort
  • Smart deduplication across growing idea bank
  • Consistent visual branding via AI image prompts
