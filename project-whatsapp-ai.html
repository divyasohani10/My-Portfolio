Overview

The Objective

This project transforms WhatsApp into a fully AI-powered conversational commerce platform. The assistant handles product discovery, shows interactive menus and carousels, tracks real-time orders via SQL queries, and manages the entire customer journey — from first message to order status — without any human agent involvement.

The Challenge

Dynamic Routing Complexity

WhatsApp's interactive components (lists, buttons, carousels) each require different API payload structures. The system needed to detect user intent from free-text AND button/list replies, route to the correct response type dynamically, and maintain conversation context — all while querying a live Cloudflare D1 SQL database for real-time order data.

The Solution

Technical Strategy

OpenAI acts as the core decision engine, converting any user message into a structured JSON object with a 'reply' and an 'action' field. n8n's Switch node reads the action and routes to the correct handler: interactive list menus for product browsing, carousel templates for product showcases, or live SQL queries to Cloudflare D1 for order tracking by Order ID or phone number. The system supports full conversation memory and handles all WhatsApp message types including text, list selections, and button replies.

Key Results

Impact

  • 24/7 automated sales and support on WhatsApp
  • Real-time order tracking via live SQL database queries
  • Interactive carousels replacing static product catalogues
  • Zero human agent required for standard customer journeys
