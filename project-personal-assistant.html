Overview

The Objective

This project turns Telegram into an intelligent personal productivity system. Send a voice message describing your workout, a task you need to remember, or an idea that just came to you — and the AI automatically transcribes it, understands the intent, and saves it to the correct Notion database. No typing, no manual logging, no friction.

The Challenge

Intent Detection from Messy Voice Input

Real voice messages are unstructured — a single message might contain a workout update, a task reminder, and a new idea all at once. The system needed to reliably detect multiple intents from one transcription, extract structured data for each (exercises, sets, reps, deadlines, idea text), and route them independently to the correct Notion databases simultaneously.

The Solution

Technical Strategy

The Telegram trigger captures incoming voice notes. n8n downloads the audio file and passes it to OpenAI Whisper for speech-to-text transcription. The transcript is then sent to a GPT-4o AI Agent configured with three tools: a workout logger, a task creator, and an idea saver. The agent autonomously decides which tools to call based on the content, extracts structured fields for each, and writes to the corresponding Notion database — all within seconds of receiving the voice message.

Key Results

Impact

  • Voice-to-Notion in under 10 seconds
  • Single message can trigger multiple simultaneous Notion updates
  • Fully hands-free personal productivity system
  • Handles workouts, tasks, and ideas from natural speech
