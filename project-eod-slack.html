EOD Email Check
& Slack Notification

n8n Gmail Google Sheets Slack Team Automation
Overview

The Objective

This workflow automatically monitors whether employees have submitted their End of Day report via email and sends a personalized Slack DM reminder to anyone who has not — without any manual checking. It connects Google Sheets as the employee database, Gmail as the detection layer, and Slack as the notification channel to automate a daily team management task completely.

The Challenge

Reliable Detection Across Multiple Employees

The system needed to batch-process multiple employees, build dynamic Gmail search queries per person, detect EOD emails using flexible keyword matching (EOD, eod, End of Day), look up each employee's Slack user ID from their email address, and send targeted DMs — all in sequence, without one failure breaking the entire batch.

The Solution

Technical Strategy

n8n reads the employee list from Google Sheets (name + email columns). A Code Node builds a dynamic Gmail search query per employee: from:[email] (EOD OR eod OR End of Day) newer_than:1d. A Split in Batches Node loops through each employee. Gmail searches for matching emails. If no EOD is found, an HTTP Request calls the Slack API to look up the user by email, retrieves their Slack user ID, and sends a personalized DM reminder. The workflow can be triggered manually or converted to a daily Cron schedule.

Key Results

Impact

  • 100% automated EOD tracking across entire team
  • Personalized Slack DM per employee — not a generic channel blast
  • Dynamic Gmail query built per person at runtime
  • Easy to extend: add Cron, change time window, update message
